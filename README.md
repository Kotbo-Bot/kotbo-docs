# Kotbo Docs

Documentation utilisateur du dashboard Kotbo, construite avec Astro et Starlight.

## Développement local

```bash
bun install
bun dev
```

Le site est disponible sur `http://localhost:4321/kotbo-docs/`.

## Production

```bash
bun run build
bun run preview
```

Le build statique est généré dans `dist/`. Chaque push sur `main` déclenche le déploiement GitHub Pages.
