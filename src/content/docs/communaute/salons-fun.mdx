<p class="doc-lead">Le module <strong>Salons Fun</strong> permet de créer des espaces ludiques et automatisés sur votre serveur Discord. Kotbo s'occupe de gérer les règles et de maintenir le score en temps réel pour plusieurs jeux classiques.</p>

:::note[Disponibilité]
Ces salons de jeux sont configurables depuis la section **Communauté → Salons Fun** de votre dashboard : [Accéder au module Salons Fun ↗](https://dash.kotbo.fr/fun).
Les modifications nécessitent les permissions d'administration de Kotbo sur le serveur.
:::

## Salon de comptage

Le jeu de comptage invite vos membres à compter le plus loin possible, dans l'ordre croissant (1, 2, 3, 4, etc.). 

### Fonctionnement et règles
* **Un chiffre à la fois** : Les utilisateurs doivent envoyer uniquement le nombre suivant. Tout message contenant autre chose ou un mauvais nombre est ignoré ou supprimé par Kotbo.
* **Pas de double envoi** : Un utilisateur ne peut pas envoyer deux nombres consécutivement. Un autre membre doit impérativement prendre la suite.

### Configuration du salon
1. Accédez à **Communauté → Salons Fun**.
2. Dans le premier bloc **Salon de comptage**, ouvrez le menu déroulant **Salon Discord** et sélectionnez le salon dédié au jeu.
3. Le bloc affiche en temps réel :
   * **Nombre** : Le score actuel de la chaîne de comptage.
   * **Dernier joueur** : L'identifiant (ID) du dernier utilisateur ayant validé un nombre.
4. Si un membre fait une erreur ou que vous souhaitez remettre la partie à zéro, cliquez sur **Réinitialiser le comptage** (un message de confirmation s'affiche).

## One Word Story

Ce jeu collaboratif invite la communauté à écrire une histoire collective, un seul mot à la fois.

### Fonctionnement et règles
* **Un seul mot par message** : Kotbo analyse chaque message envoyé. S'il contient plus d'un mot, il est supprimé automatiquement.
* **Pas d'auto-réponse** : Un membre ne peut pas envoyer deux mots de suite. Il doit attendre qu'un autre membre contribue à l'histoire.

### Configuration du salon
1. Dans le bloc central **One Word Story**, sélectionnez le salon textuel dédié à l'histoire dans le menu déroulant **Salon Discord**.
2. La zone de statut affiche l'identifiant (ID) du **Dernier auteur** ayant écrit un mot.
3. Ce salon est entièrement régulé par le bot : les mauvais messages sont supprimés et les doubles envois bloqués sans intervention des modérateurs.

## Nombre Mystère

Proposez à vos membres de deviner un nombre secret généré aléatoirement par le bot.

### Fonctionnement et règles
* Les membres proposent des nombres dans le salon dédié.
* Kotbo répond en indiquant si le nombre mystère est plus grand ou plus petit que la proposition.
* Lorsqu'un joueur trouve la bonne réponse, il gagne la partie, et le bot l'annonce sur le salon.

### Configuration du salon
1. Dans le troisième bloc **Nombre Mystère**, sélectionnez le salon textuel de jeu dans le menu déroulant **Salon Discord**.
2. Le bloc affiche la valeur du **Nombre mystère actuel** (visible uniquement par le staff sur le dashboard).
3. Pour changer la valeur cible ou lancer une nouvelle partie manuellement, cliquez sur le bouton **Générer un nouveau nombre mystère**.

## Sauvegarde des modifications

:::caution[Enregistrement requis]
Dès que vous modifiez l'affectation d'un salon fun, une barre d'action s'affiche en bas de votre écran. Vous devez cliquer sur **Enregistrer** pour que les changements soient appliqués sur votre serveur Discord. Cliquez sur **Réinitialiser** si vous souhaitez annuler vos modifications en cours.
:::
